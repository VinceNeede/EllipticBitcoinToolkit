Running the Examples
--------------------

To run these examples locally:

.. code-block:: bash

   # Navigate to examples directory
   cd examples/
   
   # Run basic usage
   python basic_usage.py
   
   # Start Jupyter for notebook examples
   jupyter notebook random_forest.ipynb
